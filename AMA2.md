# HTML and CSS Questions

## 1. What is Relative Position in CSS?
Relative position moves an element relative to its original position using top, bottom, left, or right. The element still keeps its original space in the layout.

## 2. What is the difference between Width and Max-width?
Width sets the fixed width of an element, while max-width sets the maximum width and allows the element to shrink on smaller screens.

## 3. Explain CSS Grid
CSS Grid is a layout system used to create rows and columns for designing structured and responsive web page layouts.

## 4. What is Absolute Position in CSS?
Absolute position places an element relative to its nearest positioned parent and removes it from the normal document flow.

## 5. What are Important Tags in HTML?
Some important HTML tags are `<html>`, `<head>`, `<title>`, `<body>`, `<h1>`, `<p>`, `<a>`, `<img>`, and `<div>` which help structure a webpage.

## 6. Difference Between HTML and CSS
HTML creates the structure of a webpage, while CSS is used to style and design the appearance of that webpage.

## 7. How to Center a Div in CSS?
A div can be centered by using `margin: auto` with a fixed width or by using Flexbox with `justify-content: center` and `align-items: center`.

## 8. Difference Between Ordered List and Unordered List in HTML
An ordered list (`<ol>`) displays numbered items, while an unordered list (`<ul>`) displays items with bullet points.

## 9. Advantages of Using External CSS
External CSS keeps styles in a separate file, improves code reusability, and makes website maintenance easier.

## 10. What is Inline-Block in CSS?
Inline-block allows elements to appear in one line like inline elements while still allowing width and height like block elements.

## 11. When Do We Use Sticky Position?
Sticky position is used when an element should scroll normally and then stick to a fixed position when it reaches a specified point.

## 12. What are NBSP and SP in HTML/CSS?
`&nbsp;` is a non-breaking space that prevents line breaks, while a normal space allows text to wrap to the next line.

## 13. What is the Universal Selector?
The universal selector `*` selects all elements in an HTML document and is often used for applying global styles.

## 14. What is a Background Gradient?
A background gradient creates a smooth transition between two or more colors using CSS functions like `linear-gradient()`.

## 15. Difference Between Relative and Absolute Position
Relative position moves an element from its normal position, while absolute position places an element relative to its nearest positioned ancestor.
